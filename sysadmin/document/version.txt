0. Запусть Denver

2. Подключиться по SQLyog.
3. Копировать база.
   FIND_ XML_ без данных
   *_ (остальные) c данными
   admin
4. Выполнить Скрипты по таблицам TREE_ FORM_
6. настроить коннект в     
   Z:\home\sysmaster.rr\www\Ajax\master\master_connect.php (Локальный)
   .\ru\master_connect.php (для хоста)
7. Копировать по FTP каталоги
   Z:\home\sysmaster.rr\www\sysadmin\ (убрать лишние файлы)
                            index.php без кеширования   
   Z:\home\sysmaster.rr\www\images\tree_S
   Z:\home\sysmaster.rr\www\image\top_seach\
   Z:\home\sysmaster.rr\www\style\tree.css 
   Z:\home\sysmaster.rr\www\Ajax\master\ru\master_connect.php 
   Возможно библиотеки...
8. Переименовать файл на хосте:
   TiSQL_user.php --> TiSQL_sys.php 
9. Проверить коннект на хосте .\sysadmin
   Если не коннектиться и CGI, копировать файл
